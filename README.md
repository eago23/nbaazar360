# n'Bazaar360 - Pazari i Ri Digital Platform

Platforma digjitale për Pazarin e Ri, Tiranë.

## 🚀 Setup i Shpejtë

### 1. Klono projektin
```bash
git clone https://github.com/eago23/nbazaar360.git
cd nbazaar360
```

### 2. Instalo Backend
```bash
cd nbazaar360-backend
npm install
```

### 3. Instalo Frontend
```bash
cd ../nbazaar360-frontend
npm install
```

### 4. Nis aplikacionin

**Terminal 1 - Backend:**
```bash
cd nbazaar360-backend
npm start
```

**Terminal 2 - Frontend:**
```bash
cd nbazaar360-frontend
npm run dev
```

### 5. Hap në browser
- **Frontend:** http://localhost:3000
- **Admin Login:** admin@nbazaar360.al / admin123
